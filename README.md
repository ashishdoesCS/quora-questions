# Quora Question Similarity Detection

## 🚀 Overview
This project uses NLP techniques to determine whether two questions are semantically similar or not.

## 🧠 Features
- Text preprocessing (stopwords, stemming)
- Bag of Words (BoW)
- Feature engineering
- Machine Learning models

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Streamlit (for UI)

## 📂 Project Structure
- `streamlit-app/` → Web interface
- `*.ipynb` → Model building & experiments

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py

Dataset Link - https://www.kaggle.com/c/quora-question-pairs
